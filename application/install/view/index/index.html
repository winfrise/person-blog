LeCMS-博客版是用ThinkPHP5.0 + layui2.3做的，简单方便，可用于快速搭建博客或内容管理系统。
功能:
1、内容模型有:单页模型、文章模型、链接模型、图集模型、下载模型、视频模型。
2、支持多模板切换功能。
3、支持多种存储方式，目前已支持本地和七牛云存储。后期将会扩展OSS多种存储方式。
4、文章评论(输入QQ号自动获取昵称进行评论)
5、Sitemap一键生成，有利于搜索引擎的收录
